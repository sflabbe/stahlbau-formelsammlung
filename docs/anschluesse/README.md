# Modul Anschlüsse

Dieses Verzeichnis enthält den ersten Baustein der Stahlbau-Formelsammlung.

## Geschraubte Verbindungen

Quelle: `schraubenverbindungen.tex`  
Ausgabe: `../../dist/schraubenverbindungen.pdf`

Das Modul ist bewusst als eigenständige LaTeX-Datei gehalten. Spätere Anschlussarten können in separaten Dateien ergänzt werden, zum Beispiel:

- Stirnplattenanschlüsse
- Winkelanschlüsse
- Fußplatten
- Schweißnähte
- Aluminium-Stahl-Mischanschlüsse
